# 🤖 NLP Chatbot using Hugging Face Transformers

## 📌 Objective
Build a conversational chatbot using pre-trained transformer models.

## 🛠️ Technologies Used
- Python
- Hugging Face Transformers
- PyTorch
- Jupyter Notebook

## 🚀 Model Used
- Microsoft DialoGPT (medium)

## ⚙️ Features
- Interactive chatbot
- Maintains conversation history
- Generates dynamic responses
- Exit using "exit" or "quit"

## 📊 Learning Outcomes
- Understanding transformer models
- Using Hugging Face model hub
- Text generation
- Conversational AI development
